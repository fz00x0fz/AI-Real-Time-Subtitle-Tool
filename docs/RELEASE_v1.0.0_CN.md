# AI实时字幕工具 v1.0.0 发布说明

## 🎉 首次发布

我们很高兴地宣布 **AI实时字幕工具 v1.0.0** 正式发布！这是一个功能完整、开箱即用的Windows桌面应用，可以实时捕获系统音频并通过AI转换为文字字幕。

---

## ✨ 核心功能

### 🎙️ 实时音频捕获
- 支持Windows系统音频捕获（WASAPI Loopback）
- 自动检测音频设备
- 支持立体声混音和虚拟音频设备
- 完善的外接音箱/耳机支持

### 🤖 多AI服务支持
- **OpenAI Whisper API** - 高精度语音识别
- **Azure Speech Service** - 微软语音服务
- **阿里云百炼** - 国内优化，低延迟（推荐）
- **本地Whisper模型** - 离线使用，隐私保护

### 💬 实时字幕显示
- 美观的浮动字幕窗口
- 支持窗口拖拽和调整大小
- 可调节透明度
- 自定义字体大小和颜色
- 现代化UI设计

### ⚙️ 图形化配置
- 直观的可视化配置界面
- 无需手动编辑配置文件
- 实时预览效果
- 一键保存配置

### 🔌 运行时安装本地模型
- 标准版支持按需安装本地Whisper模型
- 图形化安装界面
- 实时进度显示
- 无需重新打包

### 📦 一键打包
- 支持打包为独立exe
- 便携版支持
- 配置向导
- 开箱即用

## 🚀 快速开始

### 下载和安装

1. **下载便携版**
   - 下载 `AI实时字幕_v1.0.0_便携版.zip`
   - 解压到任意目录

2. **运行配置向导**
   - 双击 `配置向导.bat`
   - 按提示选择AI服务
   - 输入API密钥

3. **启动应用**
   - 双击 `AI实时字幕.exe`
   - 点击"▶ 开始"按钮
   - 开始使用

### 系统要求

- **操作系统**: Windows 10/11
- **内存**: 至少2GB可用内存
- **磁盘**: 至少500MB可用空间
- **网络**: 使用云端API需要网络连接

## 🐛 已知问题

### 1. 音频捕获问题

**问题**: 部分用户可能遇到"检测不到语音"

**解决方案**: 
- 查看 [音频设置指南](docs/AUDIO_SETUP.md)
- 启用立体声混音
- 使用虚拟音频设备

### 2. 外接设备支持

**问题**: 使用外接音箱/耳机时无法捕获音频

**解决方案**:
- 查看 [外接设备配置指南](docs/EXTERNAL_AUDIO_SETUP.md)
- 使用VB-Cable或Voicemeeter

### 3. 首次运行较慢

**问题**: 首次运行时启动较慢

**原因**: 
- 加载依赖库
- 检测音频设备
- 初始化UI

**说明**: 正常现象，后续运行会更快
