## 🚀 使用方法

### 打开配置界面

1. 启动AI实时字幕工具
2. 点击主窗口上的 **⚙** 按钮
3. 配置界面自动打开

### 配置步骤

1. **选择AI服务**
   - 在"AI服务"标签页选择服务类型
   - 推荐：阿里云百炼（国内用户）

2. **填写API密钥**
   - 点击"点击获取API Key"链接
   - 复制API密钥
   - 粘贴到输入框
   - 可勾选"显示密钥"查看

3. **调整参数（可选）**
   - 音频设置：调整采样率和块时长
   - 界面设置：调整窗口大小和字体

4. **保存配置**
   - 点击"保存"按钮
   - 系统自动验证配置
   - 配置保存到.env文件

5. **重启应用**
   - 关闭应用
   - 重新启动
   - 新配置生效
