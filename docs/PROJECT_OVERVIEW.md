# AI实时字幕工具 - 项目总览

## 🎯 项目简介

一个功能完整的Windows桌面AI浮窗工具，支持实时捕获系统音频并通过AI大模型转换为文字字幕显示。

**版本**: v1.2.0  
**状态**: ✅ 生产就绪  
**平台**: Windows 10/11

## ✨ 核心功能

### 1. 实时音频捕获
- ✅ Windows WASAPI Loopback
- ✅ 自动设备检测
- ✅ 立体声混音支持
- ✅ 可配置采样率和缓冲

### 2. AI语音识别
- ✅ OpenAI Whisper API
- ✅ Azure Speech Service
- ✅ 阿里云百炼（DashScope）
- ✅ 本地Whisper模型

### 3. 实时字幕显示
- ✅ PyQt5浮动窗口
- ✅ 始终置顶显示
- ✅ 可拖拽移动
- ✅ 透明度可调
- ✅ 自动隐藏

### 4. Windows应用打包
- ✅ PyInstaller一键打包
- ✅ 便携版创建
- ✅ 配置向导
- ✅ 开箱即用

## 🚀 功能特性详解

### 音频捕获
- **技术**: Windows WASAPI Loopback
- **采样率**: 8000-48000 Hz可配置
- **缓冲**: 队列管理，防止丢帧
- **设备**: 自动检测和手动指定

### AI转录
- **OpenAI**: Whisper-1模型，多语言支持
- **Azure**: 实时流式识别，低延迟
- **阿里云**: Paraformer系列，中文优化
- **本地**: 离线模型，完全免费

### 用户界面
- **框架**: PyQt5现代化设计
- **特性**: 无边框、透明、置顶
- **交互**: 拖拽、开始/停止控制
- **样式**: 自定义CSS样式表

### 应用打包
- **工具**: PyInstaller 6.3.0
- **模式**: 目录模式（推荐）
- **大小**: ~300-400 MB
- **压缩**: UPX压缩，ZIP ~150-200 MB
