1. 查看 `QUICK_FIX.md` 快速解决常见问题
2. 参考相关的 `FIX_*.md` 文档
3. 查看 `BUILD_TROUBLESHOOTING.md`

## 📝 文档维护

### 添加新文档
- 所有新文档应放在 `docs/` 目录
- 在 `README.md` 中添加链接
- 使用清晰的文件名

### 文档命名规范
- 用户指南: 大写字母，下划线分隔
- 修复文档: `FIX_` 前缀
- 故障排除: `TROUBLESHOOTING` 或 `QUICK_FIX`
- 版本发布: `RELEASE_` 或 `VERSION_` 前缀
